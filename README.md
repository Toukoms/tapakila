# Tapakila

Event web app crafted with Next.js, TypeScript, TailwindCSS, DaisyUI

## Run Locally

- Clone the project

```bash
  git clone https://github.com/Toukoms/tapakila.git
```

- Go to the project directory

```bash
  cd tapakila
```

- Install dependencies

```bash
  npm ci
```

or

```bash
  npm install
```

- Start the server

```bash
  npm run dev
```
